Module Boards.Skeletyl

This is configuration is a mimicry of BastardKB's skeletyl keyboard. (https://github.com/Bastardkb/Skeletyl).

This is is a good place to start from if you'd like to have something like the skeletyl, but with some more pinky stagger. Though, it will never be as pretty.
